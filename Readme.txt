Caesar Cipher Encryption/Decryption

This program allows you to encrypt and decrypt messages using the Caesar Cipher, a simple substitution cipher.

Features:

Encrypts and decrypts text.
User-friendly command-line interface.
Usage:

Run the executable file.
Select the desired option:
Encrypt a message
Decrypt a message
Exit
Enter the text and the shift value.
The program will display the encrypted or decrypted text.
Installation:

Download the executable file.
Run the executable file directly.
Dependencies:

Python 3
Disclaimer:

This program is for educational and demonstrative purposes only.
It should not be used for serious security applications.
The Caesar Cipher is a relatively weak encryption method.
